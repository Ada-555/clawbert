# Clawbert 🦞

**🔗 Live:** https://ada-555.github.io/clawbert/

---

**Ada's Desktop Companion** — A pixel art lobster living in a cyberpunk aquarium on your desktop.

## Features

- 🦞 **Animated pixel lobster** with CSS animations (swimming, working, thinking, sleeping, excited states)
- 💭 **Speech bubbles** with random thoughts and mood expressions
- 📊 **Status dashboard** showing mood, tasks, context, uptime
- 🫧 **Animated tank environment** with rising bubbles and data particles
- 🌙 **Day/night cycle** based on time
- 🎨 **Cyberpunk aesthetic** with neon glow and CRT scanlines
- 📱 **Responsive** — scales from mobile widget to fullscreen
- ⚡ **Pure CSS animations** — no heavy frameworks

## How to Use

Open in browser: **https://ada-555.github.io/clawbert/**

Currently running in **simulation mode** — watching Clawbert swim, change moods, and chat.
